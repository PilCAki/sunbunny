Privacy Policy

Last updated: February 2026

Summary

Sunbunny Panoramatone does not collect, transmit, or store any user data. Period.

Details

  • No telemetry: The plugin does not phone home, send analytics, or collect crash reports.
  • No network access: The plugin makes zero network connections at any time.
  • No personal data: The plugin does not read, write, or access any personal files beyond the standard VST3 plugin scan performed by your DAW.
  • No cookies: This website is static HTML hosted on GitHub Pages. It does not set cookies or use tracking scripts. GitHub’s own privacy policy applies to the hosting platform.

Third-party services

The download buttons on this site fetch release information from a local releases manifest hosted alongside the site. No external API calls are made. GitHub Pages hosts the site itself; GitHub may log your IP address per their privacy policy. We do not control or receive that data.
