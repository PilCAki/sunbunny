Installation Guide

Sunbunny Panoramatone is distributed as a VST3 plugin. Download the zip for your platform, extract it, and copy the plugin to the correct folder.

⊕ Windows

  1. Download the Windows VST3 zip from the Downloads section.
  2. Extract (right-click → Extract All…).
  3. Copy the .vst3 file (or folder) to: 
    C:\Program Files\Common Files\VST3\
  4. Open your DAW and rescan plugins:
    • Ableton Live: Options → Preferences → Plug-Ins → Rescan
    • FL Studio: Options → Manage Plugins → Start scan
    • Reaper: Options → Preferences → Plug-ins → VST → Re-scan
    • Bitwig: Settings → Plug-ins → Rescan

⌘ macOS

  1. Download the macOS VST3 zip.
  2. Double-click the zip to extract.
  3. Copy the .vst3 bundle to: 
    /Library/Audio/Plug-Ins/VST3/

    (Or for current user only: ~/Library/Audio/Plug-Ins/VST3/)

  4. If macOS blocks the file, see Gatekeeper troubleshooting.
  5. Open your DAW and rescan plugins.

🐧 Linux

  1. Download the Linux VST3 zip.
  2. Extract: 
    unzip Panoramatone_Linux_VST3_*.zip
  3. Copy the .vst3 folder to one of: 
    ~/.vst3/
# or system-wide:
/usr/lib/vst3/
  4. Rescan in your DAW (Bitwig, Reaper, Ardour, etc.).

DAW Rescan Tips

Most DAWs auto-scan the standard VST3 paths on startup. If the plugin doesn’t appear:

  • Make sure the .vst3 file/folder is in the right directory.
  • Trigger a manual rescan from your DAW’s plugin manager.
  • Restart the DAW if the rescan doesn’t pick it up.
  • Check Troubleshooting for more help.
