Verify Your Download

Compare the SHA-256 checksum of the zip you downloaded against SHA256SUMS.txt.

# macOS / Linux
shasum -a 256 Panoramatone_*.zip

# Windows (PowerShell)
Get-FileHash .\Panoramatone_*.zip -Algorithm SHA256

Trust & Safety

This plugin is an independent freeware project. The builds are currently unsigned and unnotarized, which means your OS may show a warning the first time you open it.

⊕ Windows SmartScreen

If Windows shows a “Windows protected your PC” dialog:

  1. Click More info
  2. Click Run anyway

Distributing as a .zip (not an installer) greatly reduces the chance of SmartScreen triggering.

⌘ macOS Gatekeeper

If macOS says the file “can’t be opened because it is from an unidentified developer”:

  1. Open System Settings → Privacy & Security
  2. Scroll down and click Open Anyway
